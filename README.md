
# 🍔 فست فود لاویسان - سایت منو

سایت منو آنلاین برای رستوران فست فود لاویسان

## ✨ ویژگی‌های سایت

- 🍔 **منوی متنوع**: برگر، پیتزا و ساندویچ
- 🛒 **سبد خرید**: اضافه/کم کردن آیتم‌ها
- 🎨 **طراحی زیبا**: رابط کاربری حرفه‌ای و جذاب
- 📱 **طراحی واکنش‌پذیر**: کار کردن در تمام دستگاه‌ها
- 🔍 **فیلتر دسته‌بندی**: جستجوی آسان در منو

## 🚀 شروع سریع

1. فایل `index.html` را در مرورگر باز کنید
2. منو را مرور کنید
3. کالا‌های مورد علاقه را به سبد خرید اضافه کنید
4. روی سبد خرید کلیک کنید
5. سفارش خود را تأیید کنید

## 📂 ساختار فایل‌ها

```
fastfood-lavisan/
├── index.html    # صفحه HTML
├── styles.css    # استایل و طراحی
├── script.js     # عملکردهای جاوا اسکریپت
└── README.md     # این فایل
```

## 📝 نحوه استفاده

### اضافه کردن محصول جدید
فایل `script.js` را باز کنید و به بخش `menuItems` بروید:

```javascript
{
    id: 13,
    name: 'نام محصول',
    category: 'burger', // burger, pizza, sandwich
    price: 50000,
    description: 'توضیحات محصول',
    emoji: '🍔'
}
```
